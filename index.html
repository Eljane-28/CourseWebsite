Unit 1 — Optimization

Problem

A factory produces tables and chairs. Each table requires 3 hours carpentry and 2 hours finishing; each chair 2 hours carpentry and 1 hour finishing. Weekly capacities: 120 carpentry hours, 60 finishing hours. Profit per table ₱500, per chair ₱300.

Formulation
Let x1 = # tables, x2 = # chairs
Maximize Z = 500x1 + 300x2
Subject to:
  3x1 + 2x2 ≤ 120   (carpentry)
  2x1 + 1x2 ≤ 60    (finishing)
  x1, x2 ≥ 0
Solution (Excel Solver)

Optimal solution: x1 = 20 tables, x2 = 0 chairs → Maximum profit = ₱10,000. Recommendation: produce tables only to maximize profit under given capacities.

Unit 2 — Statistics & Forecasting

Objective

Forecast monthly sales (₱) of a small store for the next 3 months using moving average and linear regression.

Data (last 6 months)

50,000; 55,000; 52,000; 60,000; 62,000; 65,000

Results

3-month moving average forecast: 59,000; 62,000; 63,000. Regression: Sales = 48,500 + 2,700 × Month, R² ≈ 0.92 (strong trend).

Midterm Project

Case: Inventory — Quick Sale Mart

Apply the EOQ model to find optimal order quantity.

Data & calculation

Annual demand D = 12,000 units; Ordering cost S = ₱500/order; Holding cost H = ₱2/unit/year.

EOQ = √(2DS/H) = √(2×12,000×500 / 2) ≈ 2,449 units.

Interpretation

Order approximately 2,449 units per order to minimize total ordering + holding costs.

Unit 4 — Decision Analysis / Game Theory

Decision tree example

Launch new product? Launch cost ₱200,000. If success (p=0.6) profit ₱500,000; if fail (p=0.4) profit ₱100,000.

EV(Launch) = 0.6(500k−200k) + 0.4(100k−200k) = ₱180,000 → Launch (EV > 0)

Game theory (toy example)

Two stores choose price High/Low. Payoff matrix: Nash equilibrium at (Low,Low) - both end up pricing low.
